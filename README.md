🧾Campus Food Management System
📌 System for managing cafeteria menu, orders, meal plans, and student consumption.
⚙️ Tecnologías
- NestJS
- MySQL
- Angular
- Postman
🚀 Funcionalidades
- Menu management
- Order processing
- Stock control
- Meal plans
- Discounts by user type
🔗 Integraciones
- Payment system (simulated)
- Student validation system
- Residence system
🛠️ Instalación
npm install
npm run start
